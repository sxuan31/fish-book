# Seafood Manager 技能指南

此技能旨在帮助你维护和扩展“海鲜视觉百科”应用。

## 核心职责
1.  **数据扩充**: 按照 `App.jsx` 中的 JSON 结构添加新的海鲜品种。
2.  **图库维护**: 使用 `scripts/` 目录下的爬虫脚本，定期从百度百科或维基百科更新本地图库。
3.  **视觉校准**: 确保新添加的图片符合“高清、实拍、主体突出”的审美要求。

## 目录说明
- `scripts/`: 存放用于自动化任务的脚本（如 `download_images.cjs`）。
- `examples/`: 存放标准的数据项 JSON 模板。
- `resources/`: 存放离线的占位图或调色盘配置。

## 使用场景示例
当用户说“添加 5 种深海鱼类”时，你应该参考 `examples/` 中的模板，并调用 `scripts/` 里的下载程序来获取真实图片。
